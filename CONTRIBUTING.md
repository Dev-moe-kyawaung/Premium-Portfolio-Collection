# Contributing

Thank you for your interest in **Moe Kyaw Aung — Premium Portfolio Collection**.

This is a personal portfolio project. External contributions are not expected,
but bug reports and suggestions are welcome.

---

## 🐛 Bug Reports

1. Go to [Issues](https://github.com/Dev-moe-kyawaung/Moe-Kyaw-Aung-Portfolio-V-8/issues)
2. Click **New Issue**
3. Use title format: `[BUG] V3 - Description of issue`
4. Include:
   - **Theme affected** (V1–V7 or main V8)
   - **Browser & version**
   - **Operating System**
   - **Steps to reproduce**
   - **Expected vs actual behaviour**
   - **Screenshot** (if applicable)

## 💡 Feature Suggestions

1. Go to [Issues](https://github.com/Dev-moe-kyawaung/Moe-Kyaw-Aung-Portfolio-V-8/issues)
2. Use title format: `[FEATURE] Brief description`
3. Describe the idea and the benefit it would bring

## 🎨 Theme Suggestions

If you have a premium design concept that would complement the existing 7 themes,
open an issue with `[THEME] Theme Name — Style Description`.

---

## Code Style (if submitting PRs)

- HTML: 2-space indent, semantic elements, `alt` on all images
- CSS: CSS custom properties for all colors, mobile-first approach
- JS: ES2020+, no frameworks, `const`/`let` only (no `var`)
- Comments: English or bilingual English/Myanmar
