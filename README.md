## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- No Node.js, npm, Python, or any other tool required

### Option 1 — Direct Open

```bash
# Clone the repository
git clone https://github.com/Dev-moe-kyawaung/Moe-Kyaw-Aung-Portfolio-V-8.git

# Navigate into folder
cd Moe-Kyaw-Aung-Portfolio-V-8

# Open any theme directly in browser
open index.html             # Main V8 (macOS)
open v1/index.html          # Obsidian Brutalist
open v2/index.html          # Aurora Luxury
open v7/index.html          # Bagan Midnight

# Windows
start index.html
# Linux
xdg-open index.html
```

### Option 2 — Local Dev Server (recommended)

```bash
# Python (built-in, no install needed)
python3 -m http.server 3000

# Node.js / npx
npx serve .

# VS Code Live Server extension
# Right-click any .html → "Open with Live Server"
```

Visit `http://localhost:3000` — navigate to `v1/`, `v2/` etc. for each theme.

## 🌐 Deployment Guide

### GitHub Pages (Recommended — Free)

**Step 1:** Push repository to GitHub

```bash
git init
git add .
git commit -m "feat: Premium Portfolio Collection — 7 themes"
git branch -M main
git remote add origin https://github.com/Dev-moe-kyawaung/Moe-Kyaw-Aung-Portfolio-V-8.git
git push -u origin main
```

**Step 2:** Enable GitHub Pages

1. Go to your repository → **Settings** → **Pages**
2. Under **Source**, select `main` branch, `/ (root)` folder
3. Click **Save**
4. Wait ~2 minutes for deployment

**Step 3:** Access live URLs

```
https://dev-moe-kyawaung.github.io/Moe-Kyaw-Aung-Portfolio-V-8/          # Main V8
https://dev-moe-kyawaung.github.io/Moe-Kyaw-Aung-Portfolio-V-8/v1/       # Obsidian
```

## 🛠 Customization Guide

### Updating Profile Information

Search and replace across all theme files:

```bash
# Name
"Moe Kyaw Aung"       → "Your Name"
"မိုးကျော်အောင်"    → "Your Myanmar Name"

# Role
"Senior Android Developer" → "Your Role"

# Location
"Tachileik, Myanmar"  → "Your City, Country"

# Email
"moekyawaung@fastmail.com" → "your@email.com"

# GitHub
"Dev-moe-kyawaung"    → "your-github-username"

# Gravatar image URL
"a2dae9a29fbf7c72..." → "your-gravatar-hash"
```

### Changing Accent Color (Example: V2 Aurora)

```css
/* Find in :root */
:root {
  --aurora1: #00d4aa;   /* Change to your colour */
  --aurora2: #7b2fff;   /* Change to your colour */
}
```

### Replacing Placeholder Images

Search for `picsum.photos` in any theme file and replace with your own image URLs:

```html
<!-- Before -->
<img src="https://picsum.photos/seed/zen1/600/450" alt="..."/>

<!-- After -->
<img src="./assets/images/your-project-screenshot.jpg" alt="..."/>
```
